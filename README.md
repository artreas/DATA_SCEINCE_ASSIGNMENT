Data Science (College Assignment)

Overview
This repository contains basic data science exercises implemented using Python. The aim of this assignment is to understand how different data science libraries such as NumPy, Pandas, Matplotlib, and Seaborn are used for data manipulation, analysis, and visualization.

Topics Covered
NumPy array manipulation
Pandas dataframe operations
Data filtering and preprocessing
Exploratory data analysis (EDA)
Data visualization using Matplotlib and Seaborn
Correlation analysis and heatmaps
ICC T20 cricket dataset analysis

Files in this Repository

NumPy_array_manipulation.ipynb
This notebook demonstrates basic NumPy operations such as creating arrays, reshaping arrays, transposing arrays, flattening arrays, inserting elements, and performing basic numerical operations.

Numpy_and_Pandas_operation_on_dataset.ipynb
This notebook focuses on Pandas dataframe operations including reading datasets, indexing and slicing, column manipulation, filtering data, handling missing values, sorting data, and applying functions using apply and lambda.

Matplotlib_and_seaborn_libraries.ipynb
This notebook contains different data visualization techniques using Matplotlib and Seaborn such as histogram plots, distribution plots, heatmaps, pair plots, box plots, and correlation analysis.

ICC_T20_Match_Analysis.ipynb
This notebook analyzes an international ICC T20 cricket dataset. The analysis includes exploring the dataset, cleaning the data, identifying top venues, analyzing team matchups, calculating win percentages, and generating match scorecards.

Datasets Used


Company sales dataset for visualization and analysis examples.
International ICC T20 match dataset for cricket match analysis.

Conclusion


This assignment demonstrates the basic workflow of data science including data manipulation, analysis, and visualization using Python libraries. It helps in understanding how raw data can be explored and analyzed to gain useful insights.
